# 📊 멀티 증권사 거래 분석 도구 (MVP)

여러 증권사 거래내역을 자동으로 통합 분석하는 Streamlit 웹앱.

## ✨ 주요 기능

- 🔄 **다중 증권사 자동 파싱**: 토스 PDF, 나무(NH) xls, (한투 양식 추가 가능)
- 💰 **종목별 손익 자동 계산**: 평균단가법, 통화별 분리
- 💎 **배당·이자·분배금 집계**: 종합소득 정산용
- 💼 **양도소득세 정산**: 개인/사업자 분기, 이월결손금 차감
- 📈 **인터랙티브 대시보드**: KPI, 차트, 필터링
- 📥 **결과 다운로드**: 엑셀 / (홈택스 양식·PDF는 추후 추가)

## 🚀 빠른 시작

### 로컬 실행

```bash
# 1. 의존성 설치
pip install -r requirements.txt

# 2. 앱 실행
streamlit run app.py

# 자동으로 http://localhost:8501 열림
```

### Streamlit Cloud 배포 (무료)

1. GitHub에 코드 푸시
2. https://share.streamlit.io 접속
3. New app → 리포지토리 선택 → 배포
4. 자동으로 URL 발급 (예: `https://yourapp.streamlit.app`)

## 📁 프로젝트 구조

```
streamlit_app/

```

## 🎯 사용 흐름

1. **사이드바**에서 증권사별 거래내역 파일 업로드 (다중 가능)
2. **신고자 유형** 선택 (개인/사업자)
3. **분석 결과** 6개 탭에서 확인:
   - 📊 대시보드 (KPI, 차트)
   - 💰 종목 손익 (평균단가법)
   - 💼 양도세 정산 (250만원 공제, 이월결손금)
   - 💎 배당·이자
   - 📋 거래내역 (필터링)
   - 📥 다운로드

## 🛠 개발 로드맵

| 단계 | 내용 | 상태 |
|---|---|---|
| MVP | 토스/나무 파서 + 기본 분석 | ✅ 완료 |
| 1차 | 한투 + 키움 + 미래에셋 파서 추가 | 🟡 외주 |
| 2차 | 홈택스 양식 자동 생성 | 🟡 외주 |
| 3차 | 회원가입 / 데이터 저장 / 결제 | 🔵 본격 SaaS |
| 4차 | B2B 회계사 모드 (다중 고객 관리) | 🔵 본격 SaaS |

## 🏢 비즈니스 모델

- **베타 (B2C)**: 일반 투자자 무료 또는 저가
- **본격 (B2B)**: 회계사·세무사 사무실 월 30~100만원
  - 다중 고객 일괄 처리
  - 홈택스 업로드 양식 자동 생성
  - 양도세 신고 시간 8시간 → 30분 단축
